# Example resources

Add local files here for runnable examples:

- `sample_story.txt` — used by `story.ts` (included).
- `reference.wav` — short reference clip for `baseten-provider.ts` (add your own).
- `sample_media.wav` — audio or video file for `audio-separation.ts` (add your own).

Paths are resolved relative to this folder from each example script.
